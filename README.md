# Musicboost-
Harmony is a modern music streaming application inspired by Spotify, built to deliver a smooth, fast, and personalized listening experience. Discover new music, stream your favorite tracks, and create playlists — all in one place.
✨ Features
🎧 Stream songs instantly
🔍 Search artists, albums, and tracks
❤️ Like and save favorite songs
📁 Create and manage playlists
🔁 Shuffle and repeat playback
🎨 Clean and modern UI
🌙 Dark mode support
📱 Responsive design
🔔 Smart recommendations
🚀 Tech Stack

Frontend: React / Flutter / Android
Backend: Node.js / Firebase
Database: MongoDB / Firebase
API: REST API
